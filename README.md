# Smart Data Analyst Agent

An intelligent, context-aware AI agent that transforms datasets into actionable insights. This project utilizes LLM reasoning, Retrieval-Augmented Generation (RAG), and modular pipelines to execute complex data analysis tasks. It features two modes of operation: a legacy Streamlit application for quick usage, and a modern architecture consisting of a FastAPI backend and a React (Vite) frontend.

## Features

- **Intelligent Query Routing**: Dynamically classifies user intent and redirects commands.
- **Automated Insight Generation**: Post-upload automatic generation of trends, outliers, and key observations.
- **Explainable AI Responses**: Provides reasoning logic.
- **Conversational Memory**: Retains the chat history in LangChain memory for multi-turn conversations.
- **Exploratory Data Analysis (EDA)**: Interactive tools for cleaning data, handling missing variables, visualizations, and generating downloadable analytical reports.

## Architecture

1. User uploads a `.csv` or `.xlsx` dataset.
2. Data is ingested, cleaned, and vectorized (FAISS vector index).
3. Query routing directs queries (e.g., filtering vs. plotting vs. analyzing).
4. Results are generated via GenAI and visualized via Plotly and React frontend.

## Technologies Used

- **Frontend**: React, Vite, TailwindCSS, Plotly.js, Lucide React (Icons).
- **Backend**: FastAPI, Pandas, Scikit-learn, LangChain, FAISS.
- **Generative AI**: Google Gemini Pro (via LangChain Google GenAI).

## Setup & Running

### Option 1: FastAPI + React Architecture (Recommended)

1. **Install Backend Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Environment Variables**:
   Create a `.env` file in the root directory and add:
   ```
   GOOGLE_API_KEY=your-google-api-key-here
   ```
3. **Start the FastAPI Server**:
   ```bash
   uvicorn api:app --reload --port 8000
   ```
4. **Start the React Frontend**:
   Open a new terminal, navigate to the `frontend` folder, and run:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

### Option 2: Streamlit Application (Legacy)

You can still run the monolithic Streamlit version of the application:
1. Ensure dependencies are installed and `.env` is setup.
2. Start the app:
   ```bash
   streamlit run app.py
   ```

## API Documentation
When running the FastAPI backend, you can visit `http://localhost:8000/docs` to view the autogenerated Swagger UI API documentation.
